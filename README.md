# blood-cell-anomaly-detection
This project analyzes blood cell data to identify and distinguish between normal and abnormal cells using statistical analysis and data visualization techniques. Key features such as circularity, chromatin density, granularity, and membrane smoothness were explored to determine their effectiveness in detecting anomalies.
